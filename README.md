A vpn made for school [fun fact: this is made at school]
# how-to-use:
# 1. 
right click setup.bat and click on run as adminstrator
# 2.
then right click start.bat and click on run as adminstrator.
your vpn is up and running.
# If you close terminal window:
just use start.bat again with the same steps above
